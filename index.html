Bootcamp Full Stack - IGTI

JavaScript + Node.js + React + MongoDB

Em maio de 2020 aceitei o desafio e iniciei a participação no Bootcamp Full Stack da IGTI. O programa tem como objetivo oferecer uma formação prática e intensiva em algumas tecnologias para desenvolvedor full stack. São 148h de aulas, em pouco mais de 2 meses.

O Bootcamp apresenta técnicas de construção de uma aplicação passando pelo back-end (Node.js e Express), front-end (vanilla JavaScript e React), persistência de dados NoSQL (MongoDB), controle de versionamento de código com Git e implantação em nuvem.

Em vez de subir separadamente ao Github todos os projetos concluídos neste bootcamp, preparei esta página e as demais como um storytelling sobre a evolução no aprendizado.

Abaixo estão listados os módulos previstos. O módulo atual é de número 1: Fundamentos. Clicando em "Detalhes", você encontra informações específicas de cada etapa, bem como acesso aos códigos e o resultado final.
