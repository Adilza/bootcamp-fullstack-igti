Fundamentos: JavaScript, HTML e CSS

Neste módulo foram apresentadas as noções de HTML e de CSS. Pude revisitar conceitos e práticas com JavaScript, além de aprender muitas coisas novas.

O destaque desta etapa foi poder criar aplicações simples, porém efetivas na fixação de conteúdo, usando vanilla JavaScript, sem utilizar frameworks ou bibliotecas desta linguagem de programação. Assim, consegui visualizar o quanto JavaScript evoluiu nos últimos anos.

Vale mencionar que na maioria das aulas foram apresentadas e utilizadas as novas formas de programar com JavaScript moderno (ES6+). Com JavaScript moderno, está prazeroso o desenvolvimento web.
