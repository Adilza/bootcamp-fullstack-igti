Bootcamp Full Stack - IGTI

Fundamentos - Trabalho Prático

Neste pequeno projeto, construí uma aplicação para visualização de cores a partir da escala RGB. Nela, pude exercitar os conceitos básicos no desenvolvimento web, com HTML, CSS e JavaScript sem bibliotecas ou frameworks.

Apesar da solicitação da IGTI ser aplicar as cores escolhidas nos ranges em um quadrado na tela, busquei trazer uma interface na qual há um rodapé com os ranges na vertical, o que permite um alcance melhor do dedo polegar em uma tela de smartphone. E a mudança de cor ocorre no background do espaço restante.

Após concluir o projeto, o refatorei para JavaScript moderno, com uso de arrow functions, variáveis const e let, entre outras mudanças.

Esta etapa foi importantíssima nos estudos do bootcamp, pois me deparei com um JavaScript mais poderoso do que presenciei em um curso passado, quando eu preferia utilizar JQuery.
